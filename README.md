# BankManager

Exercice 4 - Gestionnaire de Banque (POO Java - Héritage)

## Description
Application Java illustrant le concept d'héritage en POO.
Un compte bancaire parent est étendu par 3 types de comptes.

## Classes
- `BankAccount` : classe parente avec account et balance
- `CheckingAccount` : compte courant avec une limite
- `SavingsAccount` : compte épargne avec un taux d'intérêt
- `COD` : certificat de dépôt avec une durée
- `Main` : crée une instance de chaque classe et affiche les attributs

## Concepts POO utilisés
- Héritage (`extends`)
- Constructeurs
- Attributs et méthodes

## Comment exécuter
1. Ouvrir le projet dans IntelliJ IDEA
2. Exécuter la classe Main
3. Les informations des comptes s'affichent dans la console
